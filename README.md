# 星艦蜂群：升級戰線

太空船射擊小遊戲。核心玩法是左右橫移、射擊外星蜂群、閃避子彈，並用擊殺取得的晶片升級飛船。

## 外部試玩網址

```text
https://zathuramse.github.io/space-bee-shooter/
```

## 操作

- `A` / `←`：左移
- `D` / `→`：右移
- `Space`：射擊
- `X`：EMP 特殊攻擊
- `Z`：核彈
- `Q`：脈衝
- `C`：火焰
- `V`：雷射
- `B`：凍結
- `1-5`：升級武器、引擎、船體、護盾、僚機
- `P`：暫停
- `R`：重新開始
- 手機或平板可以使用畫面下方按鈕

## 升級系統

- 初始晶片至少 500。
- 晶片、升級等級、EMP 次數與一次性武器庫會儲存在本機進度中，重新開始後仍會累積。
- 所有升級上限放寬到 50 級。
- 武器系統：提升彈數、射速與主炮傷害。
- 引擎橫移：提升左右移動速度。
- 船體裝甲：增加最大生命。
- 能量護盾：抵擋傷害並在每波開始時回復。
- 僚機核心：自動補射側翼火力，等級越高射速、傷害與僚機數越好。

## 一次性武器與寶箱

- 核彈：清除一般敵人並重創 Boss。
- 脈衝：清除敵彈、全場傷害、補一格護盾。
- 火焰：短時間前方錐形範圍持續傷害。
- 雷射：短時間直線貫穿傷害。
- 凍結：短時間降低敵人與敵彈速度。
- EMP 與所有一次性武器都能累積次數。
- 寶箱掉落包含生命、護盾、晶片、EMP 次數與各種一次性武器。

## 體驗調整

- 每一波由 2-4 個敵群組成，不會打幾隻就結束。
- 清掉一個敵群會立即補下一批，清完整波才進入升級畫面。
- 升級成本降低，擊殺、清敵群、清波與晶片道具都會給更多晶片。
- 起始生命與護盾更寬鬆，前期敵人射擊與俯衝頻率降低。
- 擊殺會顯示分數與晶片浮字。
- 背景加入星雲、曲線光帶與更明顯的升級卡片效果。
- 攻擊新增槍口光、衝擊波、核彈波紋、脈衝波紋、火焰閃動與雷射脈動。
- 敵人的飛行路線會依波次、敵群、列與欄變化，包含 zigzag、swoop、orbit、ladder、split、rush 等模式。
- 敵人的攻擊模式也會隨關卡組合變化，不再每關固定同一種規律。

## 敵人與攻擊

- Scout：基礎直射
- Bee：瞄準玩家
- Guard：三向散射
- Sniper：高速狙擊彈
- Bomber：多發扇形彈幕
- Tank：高血量重彈
- Boss：每 5 波出現，具備血條、環形彈幕與瞄準攻擊
